markdown使用

准备

  1. 一个图床：这里推荐 聚合图床。

  2. 一个markdown写作工具：我用的是typora，但操作对新手不友好，我更推荐大家使用一些在线工具，比如markdown在线编辑器 - Markdown Editor (markdown-editor.org)。

    markdown是啥

    上网搜搜吧，看似很难，其实二十分钟准学会

    需要注意的是，所有除中文以外的符号均需在英文模式输入

    开干！

    1.标题

    markdown没有字号设置，但有标题功能，类似字号

如图，一个"#"后加一个空格就是一级标题，以此类推，点击左上角“H"快速生成

2. 特殊格式


如图，非常简单，点击左上的”B“ ”I“”S“快速设置，可叠加

但markdown不支持下划线，可用HTML语言代替

<u>这是下划线</u>

这是下划线

可见，在”“里的文字会加上下划线

3.插入图片

1）普通图片

  • 将图片上传到一个图床，至于为什么，可自行上网检索，下面以聚合图床为例展示
    聚合图床

    登陆后根据提示直接上传

    • 选择markdown--复制

  • 插入到文件

    切换回markdown编辑器，复制刚才的代码

    ![作者最棒](https://pic.imgdb.cn/item/66f898cff21886ccc0f71bb3.png)

    *注意：所有字符均为英文模式，其中，中括号里的内容为图片描述，可有可无；小括号的内容为图片链接，必须有

    Tips：不要使用在线工具自带的上传本地图片功能，应为十分不稳定，脱离本地无法查看

    3.插入其他链接（超链接）


    点击”链接“工具，在中括号中输入要显示的文字，在小括号中输入对应链接，如图

    [光宗耀组](https://director-ni.github.io/)

    即为光宗耀组，读者可直接点击查看

    这就是基本功能，更多功能如表格、图表等自己摸索就会了

    发布

    复制左侧的markdown代码，发给创作者即可（找到创作者)

    即日起至2024.10.10，投稿有随机红包返现
